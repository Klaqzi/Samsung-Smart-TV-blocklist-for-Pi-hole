# Smart TV Blocklist for Pi-hole

A simple blocklist focused on Smart TV tracking, telemetry, and ads.  
Designed for use with **Pi-hole**.

## Files
- **blocklist.txt** — machine-readable list of domains
- **blocklist-docs.md** — explanations of what each domain does and why it’s blocked

## Usage
Add the raw link of `blocklist.txt` to **Pi-hole -> Group Management -> Lists -> Add blocklist **, then update gravity

## Notes
Blocking some domains may disable features like app stores, updates, or certain streaming apps
Check the details file if something stops working
